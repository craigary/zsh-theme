## Install

First, download the theme into [`Oh My Zsh`](https://github.com/robbyrussell/oh-my-zsh)'s custom theme folder:

```
curl https://git.io/JG7mN -Lo ~/.oh-my-zsh/custom/themes/carbon.zsh-theme
```

Edit your `~/.zshrc`:

```
ZSH_THEME="carbon"
```

Then apply the theme:

```
source ~/.zshrc
```
